# 用户使用指南

## 🎮 快速开始

### 基本操作
1. **打开待办列表**: 按 `K` 键
2. **添加任务**: 输入标题 → 选择优先级 → 点击"添加"
3. **编辑任务**: 点击选中任务 → 修改信息 → 点击"编辑"
4. **完成任务**: 点击选中任务 → 点击"完成"
5. **删除任务**: 点击选中任务 → 点击"删除"

---

## 🎨 优先级系统

### 优先级类型
| 优先级 | 中文 | 颜色标识 | 输入值 |
|--------|------|----------|--------|
| LOW | 低 | 青色 | `LOW` |
| MEDIUM | 中 | 黄色 | `MEDIUM` |
| HIGH | 高 | 红色 | `HIGH` |

### 使用方法
1. **创建任务时**:
   - 在"优先级"输入框输入: `LOW`、`MEDIUM` 或 `HIGH`
   - 默认为 `MEDIUM`
   - 不区分大小写

2. **编辑任务时**:
   - 选中任务后，优先级输入框会显示当前值
   - 修改后点击"编辑"按钮保存

3. **视觉识别**:
   - 每个任务左侧有彩色条指示优先级
   - 高优先级任务显示红色条
   - 中优先级任务显示黄色条
   - 低优先级任务显示青色条

## ⚙️ 配置文件

### 配置位置
```
config/todolist.json
```

### 配置项说明

#### GUI设置
```json
{
  "gui": {
    "guiWidth": 350,           // GUI宽度
    "guiHeight": 250,          // GUI高度
    "taskListHeight": 120,     // 任务列表高度（像素）
    "enableDarkTheme": false,  // 深色主题（暂未实现）
    "hudPosition": "TOP_RIGHT", // HUD位置（暂未实现）
    "showCompletedTasks": true, // 显示已完成任务
    "sortByPriority": false    // 按优先级排序（暂未实现）
  }
}
```

#### 功能设置
```json
{
  "enableHud": true,           // 启用HUD（暂未实现）
  "enableTaskBook": true,      // 启用任务书（暂未实现）
  "enableSoundEffects": true,  // 音效（暂未实现）
  "maxTasksPerPlayer": 100,    // 每玩家最大任务数
  "autoSave": true,            // 自动保存
  "autoSaveIntervalMinutes": 5, // 自动保存间隔（分钟）
  "defaultPriority": "MEDIUM", // 默认优先级
  "enableTaskRewards": false   // 任务奖励（暂未实现）
}
```

### 调整任务列表高度

如果您想显示更多或更少的任务，可以修改配置：

1. 关闭Minecraft
2. 打开 `config/todolist.json`
3. 修改 `gui.taskListHeight` 的值：
   ```json
   "gui": {
     "taskListHeight": 150  // 增加到150像素
   }
   ```
4. 保存文件
5. 重新启动Minecraft

**推荐值**:
- 小屏幕: `100` (约4个任务)
- 默认: `120` (约5个任务)
- 大屏幕: `150` (约6个任务)
- 超大: `200` (约8个任务)

## 💾 保存功能

### 自动保存
- ✅ 关闭GUI时自动保存
- ✅ 切换筛选时自动保存
- 保存位置: `.minecraft/saves/世界名/todo/moddata.dat`

### 手动保存
- 点击"保存"按钮
- 保存后自动关闭UI
- 显示成功/失败消息

### 取消操作
- 点击"取消"按钮
- 直接关闭，不保存当前修改

## 🎯 使用技巧

### 快捷键
- `K` - 打开/关闭待办列表
- `Enter` - 快速添加任务
- `Esc` - 关闭界面

### 筛选任务
- **全部**: 显示所有任务
- **进行中**: 只显示未完成的任务
- **已完成**: 只显示已完成的任务

### 选中任务
- 点击任务即可选中（深灰色高亮）
- 选中的任务可以编辑、删除或标记完成

### 批量操作
- 暂不支持，计划在Phase 2中添加

## 🐛 常见问题

### Q: 任务数据保存在哪里？
A: 单人模式下保存在存档目录，多人模式保存在服务器文件夹。

### Q: 更换世界后任务会丢失吗？
A: 单人模式下，每个世界的任务是独立的。多人模式下，任务保存在服务器上。

### Q: 如何修改默认优先级？
A: 修改配置文件中的 `defaultPriority` 为 `LOW`、`MEDIUM` 或 `HIGH`。

### Q: 列表显示多少个任务？
A: 取决于配置文件中的 `taskListHeight`，默认120像素约显示5个任务。

### Q: 优先级有什么用？
A: 优先级用于视觉区分任务重要性。未来版本将支持按优先级排序和筛选。
