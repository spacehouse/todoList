# 配置系统使用指南

## 📋 概述

待办列表Mod现在包含一个完整的GUI配置系统，允许您在游戏中实时调整所有界面参数，无需重新编译或重启游戏。

---

## 🎮 访问配置界面

### 方法1: 通过主界面
1. 按 **K** 键打开待办列表
2. 点击右上角的 **⚙ 按钮**

### 方法2: 直接编辑配置文件
配置文件位置：`config/todolist.json`

您可以使用任何文本编辑器（如记事本、VSCode）编辑此文件，然后重启游戏。

---

## ⚙️ 可配置参数

### 📐 界面尺寸

#### GUI宽度 (guiWidth)
- **范围**: 300-800 像素
- **默认值**: 400
- **说明**: 控制整个待办列表窗口的宽度
- **推荐值**:
  - 小屏幕: 350
  - 标准: 400
  - 大屏幕: 500

#### GUI高度 (guiHeight)
- **范围**: 200-600 像素
- **默认值**: 280
- **说明**: 控制整个待办列表窗口的高度
- **推荐值**:
  - 小屏幕: 250
  - 标准: 280
  - 大屏幕: 350

#### 列表高度 (taskListHeight)
- **范围**: 50-400 像素
- **默认值**: 140
- **说明**: 任务列表区域的高度（不包括输入框和按钮）
- **影响**: 决定可以同时显示多少个任务
- **推荐值**:
  - 显示3-4个任务: 100
  - 显示5-6个任务: 140（默认）
  - 显示8-10个任务: 200

#### 任务项高度 (taskItemHeight)
- **范围**: 20-50 像素
- **默认值**: 25
- **说明**: 每个任务项的高度
- **影响**: 影响任务文本大小和间距
- **推荐值**:
  - 紧凑显示: 20-22
  - 标准: 25
  - 大字体: 30-35

### 📏 间距设置

#### 边距 (padding)
- **范围**: 0-50 像素
- **默认值**: 10
- **说明**: 窗口内容与边缘的距离
- **用途**: 增加内容周围的留白

#### 间距 (elementSpacing)
- **范围**: 0-20 像素
- **默认值**: 5
- **说明**: 元素之间的垂直间距
- **用途**: 调整输入框、按钮之间的距离

### 🎨 颜色设置

所有颜色值使用ARGB格式（8位十六进制数）：
```
格式: 0xAARRGGBB
- AA: 透明度 (00=完全透明, FF=完全不透明)
- RR: 红色分量 (00-FF)
- GG: 绿色分量 (00-FF)
- BB: 蓝色分量 (00-FF)
```

#### 常用颜色参考

| 颜色 | 代码 | 说明 |
|------|------|------|
| 纯黑 | `0xFF000000` | 默认背景 |
| 深灰 | `0xFF1A1A1A` | 任务项背景 |
| 中灰 | `0xFF333333` | 默认悬停 |
| 浅灰 | `0xFF555555` | 默认选中 |
| 深蓝 | `0xFF1E3A5F` | 现代风格背景 |
| 深紫 | `0xFF2D1B4E` | 暗色主题 |
| 深绿 | `0xFF1B4D2D` | 自然风格 |

#### 背景颜色 (backgroundColor)
- **默认值**: `0xFF000000` (纯黑)
- **说明**: 任务列表区域的背景色
- **推荐方案**:
  - 黑色主题: `0xFF000000`
  - 深灰主题: `0xFF121212`
  - 深蓝主题: `0xFF0D1B2A`

#### 选中颜色 (selectedBackgroundColor)
- **默认值**: `0xFF555555` (浅灰)
- **说明**: 被选中任务的背景色
- **推荐方案**:
  - 高对比: `0xFF4A90E2` (蓝色)
  - 柔和: `0xFF6B7B8C` (灰蓝)
  - 标准: `0xFF555555` (灰色)

#### 悬停颜色 (hoveredBackgroundColor)
- **默认值**: `0xFF333333` (中灰)
- **说明**: 鼠标悬停时任务的背景色
- **推荐方案**:
  - 高对比: `0xFF357ABD` (浅蓝)
  - 柔和: `0xFF4A5568` (灰蓝)
  - 标准: `0xFF333333` (灰色)

#### 边框颜色 (borderColor)
- **默认值**: `0xFF3F3F3F` (灰色)
- **说明**: 窗口和元素边框颜色
- **推荐方案**:
  - 细腻: `0xFF2D3748` (深灰蓝)
  - 明显: `0xFF4A5568` (中灰蓝)
  - 标准: `0xFF3F3F3F` (灰色)

## 🎯 使用场景

### 场景1: 列表显示的任务太少

**问题**: 只能看到3-4个任务，需要频繁滚动

**解决方案**:
1. 打开配置界面 (⚙)
2. 增加"列表高度"到 180-200
3. 可选：减小"任务项高度"到 22
4. 点击"保存并应用"

**效果**: 可以同时显示7-8个任务

---

### 场景2: 字体太小看不清

**问题**: 任务标题文字太小

**解决方案**:
1. 打开配置界面 (⚙)
2. 增加"任务项高度"到 30-35
3. 点击"保存并应用"

**效果**: 任务项变大，文字更清晰

---

### 场景3: 界面太拥挤

**问题**: 按钮、输入框挤在一起

**解决方案**:
1. 打开配置界面 (⚙)
2. 增加"GUI高度"到 320-350
3. 增加"间距"到 8-10
4. 点击"保存并应用"

**效果**: 界面更宽敞舒适

---

### 场景4: 想要暗色主题

**问题**: 默认的黑色主题太单调

**解决方案**:
1. 打开配置界面 (⚙)
2. 修改颜色值:
   - 背景颜色: `0xFF0D1B2A` (深蓝)
   - 选中颜色: `0xFF1E3A5F` (蓝灰)
   - 悬停颜色: `0xFF2D4A6F` (浅蓝灰)
   - 边框颜色: `0xFF1E3A5F` (蓝灰)
3. 点击"保存并应用"

**效果**: 现代化的深蓝主题

---

### 场景5: 窗口太大超出屏幕

**问题**: 在小屏幕上无法看到完整的窗口

**解决方案**:
1. 打开配置界面 (⚙)
2. 减小"GUI宽度"到 320-350
3. 减小"GUI高度"到 240-260
4. 减小"列表高度"到 100-120
5. 点击"保存并应用"

**效果**: 窗口适应小屏幕

## 🔧 高级技巧

### 技巧1: 测试不同配置

配置界面支持实时预览：
1. 修改参数
2. 点击"保存并应用"立即查看效果
3. 如果不满意，按 ESC 返回配置界面继续调整

### 技巧2: 快速重置

如果配置混乱，一键恢复默认：
1. 打开配置界面
2. 点击"重置默认"
3. 所有参数恢复到初始值

### 技巧3: 手动编辑配置文件

对于精确的颜色配置，建议直接编辑 `config/todolist.json`：

```json
{
  "gui": {
    "backgroundColor": 4278190080,
    "selectedBackgroundColor": 4290822330,
    "hoveredBackgroundColor": 4286515715,
    "borderColor": 4286515715
  }
}
```

**注意**: JSON中的颜色是十进制整数，需要从十六进制转换。

### 技巧4: 备份配置

在调整配置前，备份配置文件：
```bash
# Windows
copy config\todolist.json config\todolist.json.backup

# Linux/Mac
cp config/todolist.json config/todolist.json.backup
```

## 🐛 故障排除

### Q: 修改配置后没有效果？

**A**: 确保点击了"保存并应用"按钮，然后关闭并重新打开待办列表界面。

### Q: 配置界面显示不全？

**A**: 尝试先手动编辑 `config/todolist.json`，减小GUI尺寸：
```json
{
  "gui": {
    "guiWidth": 400,
    "guiHeight": 280
  }
}
```

### Q: 颜色输入显示"无效"？

**A**: 确保颜色格式正确：
- 必须以 `0x` 开头
- 必须是8位十六进制数
- 示例：`0xFF000000` ✓
- 示例：`FF0000` ✗ (缺少0x前缀)
- 示例：`0x000000` ✗ (缺少透明度通道)

### Q: 如何查看当前配置？

**A**: 打开 `config/todolist.json` 文件，所有当前配置都在里面。
