# Changelog

所有重大变更会记录在此文件中。

日期格式：`YYYY-MM-DD`

## [1.0.0] - 2026-02-21

### 能力概览
- **任务管理**：在游戏内 GUI 中创建/编辑/删除/完成任务；支持优先级（低/中/高）、标签、搜索与筛选（状态/优先级/文本/标签）。
- **项目维度**：支持个人项目与团队项目；侧边栏项目列表/搜索与项目切换；支持新建、编辑、删除项目，并提供团队项目成员管理与加入申请。
- **HUD 展示与配置**：右上角 HUD 显示当前视图任务列表，支持展开/收起与数量提示；单人世界 HUD 固定为个人视图并隐藏团队视图；提供 HUD 配置界面（尺寸、显示条数、默认视图、空列表显示、位置拖拽预览等）。
- **多人协作**：多人下个人任务按玩家独立保存；团队任务在服务器侧共享并同步给玩家；团队视图改动需保存提交，取消/关闭会丢弃本地改动并从服务器刷新，减少冲突。
- **指派与执行流**：团队任务支持领取/放弃/指派与“分配给我”等视图；离线玩家的指派人名称可回退显示，保证可读性。
- **权限与审计**：服务端权限中心基于角色、视图范围、完成状态与指派关系判定团队操作；所有生效的团队操作写入统一操作日志（`[TEAM_OP]`）。
- **持久化与多语言**：任务与项目数据以文件持久化（单人/多人均支持）；界面与文档提供中英文版本。
