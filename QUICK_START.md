# 快速开始：配置系统

## 🎯 30秒上手

### 第一步：打开配置
1. 按 **K** 键打开待办列表
2. 点击右上角的 **⚙** 按钮

### 第二步：调整配置
在配置界面中，您可以修改：

#### 📐 调整尺寸
- **GUI宽度/高度** - 窗口大小
- **列表高度** - 显示多少个任务
- **任务项高度** - 每个任务的大小

#### 📏 调整间距
- **边距** - 内容到边缘的距离
- **间距** - 元素之间的距离

#### 🎨 调整颜色
- **背景颜色** - 主背景色
- **选中颜色** - 选中任务的背景
- **悬停颜色** - 鼠标悬停时的颜色
- **边框颜色** - 边框线条颜色

### 第三步：应用配置
- 点击 **"保存并应用"** - 立即查看效果
- 点击 **"重置默认"** - 恢复初始设置
- 点击 **"取消"** - 放弃修改

---

## 🎨 颜色格式

所有颜色使用8位十六进制格式：`0xAARRGGBB`

### 常用颜色
```
黑色:   0xFF000000
白色:   0xFFFFFFFF
红色:   0xFFFF0000
绿色:   0xFF00FF00
蓝色:   0xFF0000FF
灰色:   0xFF888888
深灰:   0xFF333333
```

### 快速配色方案

#### 🔵 蓝色主题

## 💡 常见调整

### "我想看到更多任务"
```
列表高度: 180
任务项高度: 22
```

### "界面太挤了"
```
GUI高度: 350
间距: 8
```

### "字体太小"
```
任务项高度: 32
```

### "窗口太大"
```
GUI宽度: 320
GUI高度: 240
```
